
# Sistema de Despachante - Consulta Veículo DETRAN RJ

Projeto básico em Python para consulta de veículos.

## 1 - Instalar Python
https://www.python.org/downloads/

## 2 - Baixar ChromeDriver
https://chromedriver.chromium.org/downloads

Coloque o chromedriver na mesma pasta do projeto.

## 3 - Instalar dependências

Abra o terminal na pasta do projeto:

pip install -r requirements.txt

## 4 - Rodar o sistema

python app.py

## 5 - Abrir no navegador

http://localhost:5000

## Observação

Este projeto é um modelo inicial.
A automação do DETRAN pode precisar ajustes dependendo das mudanças no site.
